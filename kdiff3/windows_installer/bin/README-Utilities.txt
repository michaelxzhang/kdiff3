README for Utilities

Utilities in this directory:

- GNU sed, diff, diff3, sdiff, and cmp utilities.
  The files are not required for running KDiff3 but provided for convenience.

  The binaries were extracted from packages available here:
  http://gnuwin32.sourceforge.net/packages/sed.htm
  http://gnuwin32.sourceforge.net/packages/diffutils.htm

  The original packages are available here:
  http://www.gnu.org/software/sed/sed.html 
  http://ftp.gnu.org/gnu/sed

  http://www.gnu.org/software/diffutils/diffutils.html
  http://ftp.gnu.org/gnu/diffutils/
